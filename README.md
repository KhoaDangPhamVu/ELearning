# Project E-Learning help student practice English by ChayDoAn

## Installation
**1. Clone project**
```
git clone https://github.com/KhoaDangPhamVu/ELearning.git
```
**2. In BackEnd Folder**
```
cd BackEnd
npm install
node server.js
```
**#3. In FrontEnd Folder**

```
cd FrontEnd
npm install
npm run dev
```
**Admin Account**

```
username: admin
password: admin123
```
